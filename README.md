# Nyhetskartan 🌍

Interaktiv världskarta med dagens viktigaste nyheter, AI-assisterad analys och källtransparens.

## Features

- 🗺️ MapLibre-baserad interaktiv världskarta
- 🔴 Pulserande röda markörer för nyheter (större = viktigare)
- 📊 Tier 1/Tier 2-klassificering baserad på AI-scoring
- 🤖 gAIa-syntes med AI-genererad analys
- 📰 Flerkällverifiering + källtransparens

## Live Demo

[Öppna Nyhetskartan](https://hakankarlsson.github.io/nyhetskartan/)

## Datakällor

- SVT Nyheter (RSS)
- Sveriges Radio (RSS)  
- GNews API
- Nominatim (geocoding)
